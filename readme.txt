Pixel Arpeggiator
Tile-Pattern-Based Pixel Paint Apllication
---------------------------------------------
筆触分割・CRTディスプレイにおけるドットにじみなどから着想を得たペイントソフト


painting software 
inspired by divided brushstroke 
and the color bleeding of CRT display.
---------------------------------------------
3×3タイルのカラーパターンと配置を設定し、最大64個までパレットに保存できます。
画像の進行状況とパレットの状態を.datファイルに保存し、そこから再開できます。
PNG,JPEG出力に対応しています。
（※保存フォルダに日本語が含まれていると保存・読み込みができません）

You can set the color pattern and arrangement on a 3*3 tile and save up to 64 of them in the palette.
Save your image progress and palette state to a .dat file and resume from there.
(※If the save folder contains Japanese, saving and loading will not be possible.)






2025/08/14 構成色を16色（グローバルパレット）に限定
